# Customer-Behavior-Analysis
This project analyses 1,000 customer records to identify patterns in purchasing behaviour, revenue distribution, customer satisfaction, and regional performance. The analysis was performed using Microsoft Excel to derive actionable business insights across demographics, product categories, geography, and behavioural metrics.

# Dataset 

<a href="https://github.com/kh83harsh-afk/E-Commerce-Customer-Behavior-Analysis/blob/main/Dataset.xlsx">Dataset</a>

The dataset contains 1,000 rows and the following columns:
### Customer Demographics
•	Customer_ID
•	Age
•	Year of Birth
•	Generation
•	Gender
•	Income_Level
•	Marital_Status
•	Education_Level
•	Occupation
•	City_Name
•	State
•	Country

### Purchase Details
•	Purchase_Category
•	Purchase_Amount
•	Frequency_of_Purchase
•	Return_Rate
•	Purchase_Channel
•	Device_Used_for_Shopping
•	Payment_Method
•	Time_of_Purchase
•	Discount_Used
•	Shipping_Preference

### Behavioral & Engagement Metrics
•	Brand_Loyalty
•	Product_Rating
•	Time_Spent_on_Product_Research (hours)
•	Social_Media_Influence
•	Discount_Sensitivity
•	Customer_Satisfaction
•	Engagement_with_Ads
•	Customer_Loyalty_Program_Member
•	Purchase_Intent
•	Time_to_Decision
